🍽️ nmnm | Luxury Restaurant Platform

A sophisticated digital storefront, online ordering system, and table reservation platform with a premium dark and orange-red theme.

💼

Freelance Project

This platform was custom-built for a client to digitize their restaurant operations, providing both an elegant customer-facing ordering system and an administrative dashboard.

React (Vite)Node.jsExpressMongoDBMongooseJWT AuthenticationVanilla CSSLucide Icons

Built as a freelance solution, this platform features an immersive dark/orange-red design system designed to wow guests. Customers can reserve seats, browse catalog items across multiple categories, customize orders, and track preparation statuses. Staff and administrators can handle orders, seatings, and menu management in real-time.

🚀 Key Features

  • Elegant Storefront & Booking: Interactive landing page, testimonials, opening schedules, and a quick table reservation form.
  • Menu & Customization: Multi-category menu with real-time search, filters, and custom item choice modals.
  • Persistent Cart: Keeps item selections and updates order details locally.
  • Order Tracking: A live preparing status visual tracker (Pending ➔ Preparing ➔ Out for Delivery ➔ Completed).
  • Staff & Admin Portals: Secure interface for managing menu items (CRUD), reservations, and live order queues.

🗄 Mongoose Schemas

  • User: Name, Email, Password, Role (Admin/Staff/Client).
  • MenuItem: Name, Description, Price, Image, Category, Ingredients, Availability.
  • Order: Items, Total Price, Delivery Details, Status, Date.
  • Booking: Guest Name, Date, Time, Guests Count, Status (Confirmed/Pending).
