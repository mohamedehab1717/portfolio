Al Yaqout is a sophisticated Shopify store designed for a luxury Middle Eastern brand. The project required a deep dive into Shopify's Liquid engine to build a custom shopping experience that handles complex product variants and provides a seamless RTL (Right-to-Left) interface.

🚀 Core Features

  • Custom Product Logic: Advanced "bead" selection system for personalized items.
  • RTL Excellence: Fully mirrored layout for Arabic language support without breaking design integrity.
  • Premium UI: Clean, minimalist aesthetic with high-quality transitions and micro-interactions.
  • Responsive Filtering: Fast, custom-coded filters for navigating large product collections.

🛠 Development

The store was built using a custom Shopify theme approach, focusing on performance and mobile-first experience. I implemented custom Liquid sections to allow the client to easily manage content through the Shopify Customizer.
