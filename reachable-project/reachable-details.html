♿ Reachable Full Stack Project

A full-stack web platform designed to help people with disabilities discover accessible places and services in Egypt.

PHPLaravelMySQLLaravel SanctumComposerHTML5CSSVanilla JSBlade (Laravel)WAMP ServerLaravel MixAxiosGit

Reachable is a centralized directory solving the struggle of finding verified accessible public places. I designed and built the complete Laravel MVC backend, relational database schema, and all frontend pages. The platform categorizes locations by type, helping users find hospitals, malls, restaurants, schools, public transport, adaptive sports, job opportunities, and medical supply shops tailored to their accessibility needs.

🚀 Features

👤 User Features

  • 8 Category Sections (Hospitals, Malls, Jobs, etc.)
  • Authentication System (Register/Login via Auth)
  • Job Filtering by Disability Type (mobility, hand, speech, vision)
  • Feedback & Rating System with validation
  • Video Hero Banner & Smooth AOS animations

🛠 Admin Features

  • Backend admin area for managing categories
  • Content management and auditing
