## ✨ Features

- **3D Interactive Hero:** A CSS-native rotating 3D cube with orbiting rings and floating shapes.
- **Starfield Background:** High-performance HTML5 Canvas particle system with dynamic constellation mapping.
- **Glassmorphism UI:** Sophisticated design language using `backdrop-filter` and translucent surfaces.
- **3D Tilt Effects:** Interactive card hover effects using custom JavaScript tracking.
- **Responsive & Fluid:** Fully optimized for all devices, from mobile phones to ultra-wide monitors.
- **Scroll Reveal Engine:** Staggered animations powered by Intersection Observer API.
- **Dark Mode Aesthetic:** Premium space-themed color palette with primary/accent glow variants.
