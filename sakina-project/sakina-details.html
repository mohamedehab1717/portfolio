🧠 Sakina | Mental Health Support Platform

An AI-powered mental wellness companion designed to provide emotional support, mood tracking, guided exercises, and therapeutic interventions.

🎓

Graduation Project

This platform was developed as a graduation project. A comprehensive mental health support system combining modern web technologies with AI-powered emotional intelligence.

👨‍💻

My Role — Full Stack Developer: I was responsible for building the entire application from the ground up — frontend UI/UX with React, backend APIs with Node.js & Express, database design with MongoDB, authentication system, real-time features, and all application logic. The only part I did not work on was the AI/ML module, which was handled by a teammate.

React (Vite)Node.jsExpressMongoDBMongooseJWT AuthenticationTailwind CSSZustandFramer MotionRechartsREST APIsMulter

Sakina is a full-stack mental health web application that combines an empathetic AI chatbot with a rich set of self-care tools. Users can track their moods, journal their thoughts, complete guided therapeutic exercises, listen to calming music therapy sessions, and receive personalized risk assessments — all within a beautiful, calming interface designed to feel safe and supportive.

🚀 Key Features

💬
AI Chat Companion An empathetic chatbot with voice input support, emotional analysis, and context-aware conversations.
📊
Mood Tracker Interactive mood logging with visual analytics, emotional shift detection, and historical charts via Recharts.
📓
Smart Journal A personal journaling system for self-reflection with timestamps and mood correlation.
🧘
Guided Exercises Breathing exercises, grounding techniques, and mindfulness activities with step-by-step guidance.
🎵
Music Therapy Curated calming audio sessions with a built-in player using Howler.js for ambient sound therapy.
⚠️
Risk Assessment Survey-based mental health evaluations with scoring algorithms and intervention recommendations.
🛡️
Crisis Support (Back to Safe) Emergency grounding toolkit with instant access to calming resources and safety plans.
📈
Recovery Journey Personalized progress tracking with milestones, emotional shift analytics, and insights dashboard.

🗄 Mongoose Schemas

  • User: Name, Email, Password (bcrypt), Profile, Preferences.
  • Mood: Emotion type, Intensity, Notes, Timestamps.
  • JournalEntry: Content, Tags, Mood correlation.
  • Conversation & Message: Chat history, AI responses, Emotional context.
  • Exercise: Type, Steps, Duration, Category.
  • Survey: Questions, Scores, Risk levels.
  • InterventionSession: Therapy type, Progress, Outcomes.
  • Event: Calendar events, Reminders.
  • UserInsight: Patterns, Emotional trends, AI-generated insights.

⚙️ Technical Highlights

  • JWT-based authentication with protected routes
  • Zustand for lightweight state management
  • Framer Motion animations throughout the UI
  • Recharts for mood analytics & data visualization
  • Multer for file/audio uploads
  • Rate limiting & Helmet for API security
  • Adaptive UI with theme provider
  • Admin dashboard for content management
  • Password recovery via Nodemailer
  • Responsive mobile-first design
  • Onboarding flow for new users
  • Real-time system health monitoring
