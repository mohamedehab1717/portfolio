# 📝 Task Manager Pro

> Full Stack MERN Application

A high-performance task management application built with the MERN stack. Features project grouping, JWT authentication, real-time deadlines, and a sleek glassmorphism UI design.

---

## ✨ Key Features

- **Task CRUD** — Create, read, update, and delete tasks with ease
- **Project Grouping** — Organize tasks into projects for better workflow management
- **JWT Authentication** — Secure user registration and login
- **Real-time Deadlines** — Track task deadlines with visual indicators
- **Glassmorphism UI** — Modern, sleek design with frosted-glass effects
- **Responsive Design** — Fully optimized for desktop and mobile

---

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | React, CSS3 (Glassmorphism) |
| Backend | Node.js, Express |
| Database | MongoDB, Mongoose |
| Auth | JWT, Bcrypt |

## 🚀 Getting Started

```bash
git clone https://github.com/mohamedehab1717/Task-Manager.git
cd Task-Manager

# Start backend
cd backend && npm install && npm run dev

# Start frontend
cd ../frontend && npm install && npm run dev
```
