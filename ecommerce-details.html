📦 E-Commerce Full Stack Project

A comprehensive, production-ready storefront built for scale and premium user experience.

Node.jsExpressMySQLJWTbcryptjsCORSVanilla JS HTML CSSRender (Deployment) Postman (API testing)VS Code

Built a full-stack e-commerce platform with user authentication, product management, and order processing. Implemented REST APIs using Node.js and Express, integrated MySQL database, and developed the frontend using HTML, CSS, and JavaScript. The project also includes an admin dashboard for managing products.

🚀 Features

👤 User Features

  • User registration and login (JWT AUTH)
  • Browse products & interactive cart
  • Checkout and secure order placement

🛠 Admin Features

  • Add, Edit, and Delete products
  • Manage inventory in real-time

📂 Structure & Database

e-commerce-project
├── routes (users, products, orders)

🗄 Database Main Tables

  • Users, Products, Orders, Order_Items
  • Secure bcrypt hashing for passwords
